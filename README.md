# architecture-fe

a boilerplate for FE architecture

一个**前端工程化脚手架**样板

> 个别文件内有注释
# command

### dev
```
npm run dev
open localhost:9000
```
# main functionalities 

- npm scripts [ dev开发/product生产/deploy发布 ]

- babel [ ES6->ES5->ES3 ]

- webpack [ img/css/js 等处理 ]

- jest [ 单元测试 ]

- vue/react [ 框架 ]

# file structure
```
|- src
|- |- app
|- |- components 
|- |- global
|- |- |- reset.css
```
